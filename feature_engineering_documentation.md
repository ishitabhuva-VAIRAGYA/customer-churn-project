# Feature Engineering Documentation

Feature engineering plays an important role in improving machine learning model performance.

This project created several meaningful features from the existing dataset.

---

## Engineered Features

### 1 Customer Lifetime Value (CLV)

Formula:

CLV = MonthlyCharges × Tenure

Purpose:
Estimate the total value generated by a customer.

---

### 2 Average Spend

Formula:

AverageSpend = MonthlyCharges / Tenure

Purpose:
Understand spending behavior.

---

### 3 Tenure Group

Customers were grouped based on tenure duration:

- New Customers
- 1 Year Customers
- 2 Year Customers
- Long Term Customers

Purpose:
Identify churn patterns across customer lifetime stages.

---

### 4 Charges Per Tenure

Formula:

ChargesPerTenure = MonthlyCharges / Tenure

Purpose:
Measure spending intensity.

---

### 5 Long-Term Customer Indicator

Binary feature:

1 → Tenure > 24 months  
0 → Tenure ≤ 24 months

Purpose:
Identify loyal customers.

---

## Impact of Feature Engineering

The engineered features provide deeper insights into customer behavior and improve the predictive ability of the machine learning model.
